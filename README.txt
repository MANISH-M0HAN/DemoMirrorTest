## Installation
1. **Download Files**: Download all the files in this folder.

2. **Download Additional File**: Inside the `link.txt` file, there is a URL that links to an additional file necessary for the application to run. Download this file as well.

3. **Place Files in Same Directory**: After downloading, place all the files, including the one from the link, in the same directory.

4. **Run the Application**: Open a terminal in the directory where all the files are located and run the following command:
